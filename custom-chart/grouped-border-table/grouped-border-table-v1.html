Grouped Border Table

v1 — open in browser to compare variants

Context: A performance table where rows are grouped by a shared dimension value (e.g. OS = Android / iOS). Each group is visually enclosed so readers instantly see which rows belong together — without merging cells or pivoting. Conditional cell coloring highlights outliers. Requested when Holistics DataTable grouping alone isn't visually strong enough.

Variant 1 — Blue Panel Border

⭐ Recommended

Each group is enclosed in a 2px blue border box. The column header sits above all groups. A 10 px gap separates groups. Conditional coloring highlights CPDAU and ROI outliers. Closest match to the reference screenshot.
